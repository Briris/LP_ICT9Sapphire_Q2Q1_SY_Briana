Name Bitcoin

Description The public blockchain used to create and manage the cryptocurrency of the same name.
Bitcoin mining is the race between miners to hash specific values and other block information to find the solution to a hashing problem and add a block to the blockchain.
The winning miner is rewarded with bitcoins.
